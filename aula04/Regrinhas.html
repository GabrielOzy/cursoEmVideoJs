Regras JS, anotações

  1. O sinal "=" no JS significa "Recebe"
  2. O Sinal de "+" tem a função de Concatenação
  3. O comando "prompt" tem uma característica de sempre retornar uma string
  4. Para converter uma string para número eu posso usar (Number()), (Number.parseInt(n)), (Numer,parseFloat(n))
  5. Para converser um número para uma string eu posso usar (String(n), (n.toString())

Identificadores

  • Podem começar com letra,$ ou _
  • Não podem começar com números
  • É possível usar Letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Formatando Strings

var s = ´JavaScript´

  • 'Eu estou aprendendo s' // Não faz interpolação>
  • 'Eu estou aprendendo' + s // usa Concatenação
  • ´Eu estou aprendendo ${s}´ // Usa template string
  • s.length // Quantos caracteres a string tem
  • s.toUpperCase() // Tudo para 'MAIÚSCULAS'
  • s.toLowerCase() // tudo para 'minúsculas'

Formatando Números

var n1 = 1543.5

  • n1.toFixed(2) // Ele me mostra 2 casas decimais, no caso 1543.50
  • n1.toFixed(2).replace('.', ',') // eu troco o . por ,
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})
